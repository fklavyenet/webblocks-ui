Session 1 foundation

A playful product layer inside WebBlocks, not a copy of the admin UI.

WebGames UI keeps WebBlocks spacing, token discipline, and structure, then adds brighter energy, softer surfaces, bigger touch targets, and stronger active states for lightweight browser games.

WebGames UI foundation

Build game screens with shared DNA and a different mood.

This layer is mobile-first, touch-friendly, colorful, and clean. It is meant to wrap future games without pulling them toward dashboard or baby-app styling.
