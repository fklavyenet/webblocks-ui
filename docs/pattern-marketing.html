Pattern example: marketing

Marketing pattern example

Use calmer rhythm when the page is public, editorial, or promotional.

Marketing surfaces should not inherit the density or chrome of an admin dashboard. Use broader spacing, stronger type hierarchy, and clearer calls to action.

Start with authRead content shell

Pattern-first still applies

Marketing is still a pattern choice. The page should feel different from a dashboard before any individual card or button is inspected.

Shipped surfaces and primitives remain enough

Buttons are primitives. Cards and promo blocks are surfaces. Icons stay in the asset layer. The difference is in the shell, spacing, and page rhythm.

Public hierarchy

Keep page sections more editorial and less operational.

Clear message

Lead with proposition and action instead of dense controls.

Theme still ships

Accent, preset, and font axes can change tone without custom CSS.
