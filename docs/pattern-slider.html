Slider

wb-slider is the canonical track-based media and content carousel pattern. It fills the box it is placed in, keeps media and content as separate layers, and supports arrows, dots, keyboard, swipe, loop, and optional autoplay through shipped JavaScript.

Track-based movementMedia/content layersWBSlider runtime
Pattern boundary Use wb-slider for hero sliders, split media/content sections, and contained carousels. Use wb-gallery for equal-tile image collections and modal viewing.

Core contract

The authored contract is explicit: a slider root, a viewport, a track, one or more slides, optional media, a content slot, and optional controls. The JavaScript auto-creates dot buttons when the dots container is empty.

<div class="wb-slider wb-slider-height-lg" data-wb-slider>
  <div class="wb-slider-viewport">
    <div class="wb-slider-track">
      <article class="wb-slide">
        <img class="wb-slide-media" src="..." alt="">
        <div class="wb-slide-content">...</div>
      </article>
    </div>
  </div>
  <div class="wb-slider-controls">
    <button class="wb-btn wb-btn-icon wb-slider-arrow wb-slider-prev" data-wb-slider-prev aria-label="Previous slide">...</button>
    <div class="wb-slider-dots" data-wb-slider-dots></div>
    <button class="wb-btn wb-btn-icon wb-slider-arrow wb-slider-next" data-wb-slider-next aria-label="Next slide">...</button>
  </div>
</div>

Behavior hooks

HookSupportNotes
data-wb-sliderYesOpts the root into the shipped runtime.
data-wb-slider-prev / data-wb-slider-nextYesArrow controls scoped to the nearest slider.
data-wb-slider-dotsYesRuntime creates dot buttons when the container is empty.
data-wb-slider-autoplayYesDisabled by default. Use with data-wb-slider-interval.
data-wb-slider-loopYesEnabled by default.
data-wb-slider-keyboardYesArrow keys work when focus is inside the slider.
data-wb-slider-swipeYesPointer swipe is enabled by default.

Composition guidance

Let the placed container decide whether the slider is full viewport, split-section, or card-contained. The slider itself fills its own box; it should not force full-bleed page layout unless a height or ratio modifier asks for it.

Media rule Prefer real img or picture elements with wb-slide-media. object-fit: cover gives background-like coverage while preserving alt text, loading, and responsive image support.
