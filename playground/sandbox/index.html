AI workflow and docs

Using AI-generated HTML? Before trusting the result, confirm that the snippet uses real wb-* classes, sticks to shipped primitives or patterns, and stays inside HTML-only scope.
Check the class names Prefer real shipped wb-* classes and simple semantic HTML over invented wrappers or helper vocabularies.
Use shipped patterns when possible If the markup is trying to approximate an auth shell, hero, or dashboard frame, compare it against the matching docs example instead of layering more custom structure.
External assets are blocked here Remote images, stylesheets, scripts, and embeds from pasted HTML are intentionally blocked, and custom CSS or JavaScript are still out of scope in Playground V2.
